# WMH1.0 H5

这是从 `WMH1.html` 拆出来的 H5 前端骨架。

## 技术栈

- `Vue 3`
- `Vite`
- `TypeScript`
- `Pinia`
- `Vue Router`
- `Tailwind CSS`
- `Axios`
- `lucide-vue-next`

## 本地运行

```bash
cd apps/h5
npm run dev
```

## 构建

```bash
cd apps/h5
npm run build
```

## 环境变量

复制 `./.env.example`，然后按自己的后端地址修改 `VITE_API_BASE_URL`。
