# 协作约定

这个仓库默认按中文协作。

## 回复规则
- 每次回复都要说明本次做了什么。
- 每次回复都要说明为什么这么做。
- 每次回复都要说明当前计划和下一步要做什么。
- 每次新一轮工作开始时，先给出一个大致的 token 和时间估计，方便你安排回来查看电脑的时间。

## 记录规则
- 优先查看 `计划.md` 和 `更新记录.md`。
- 每次有新的进展，就同步更新这两个文件。
- `计划.md` 负责记录当前状态、当前目标和下一步。
- `更新记录.md` 负责记录每一次变更的时间、内容、原因和后续动作。
